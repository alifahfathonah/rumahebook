# RUMAHEBOOK
this is a simple website for downloading ebooks. I made this web for my friend who loves to read pirated books. This website is made with procedural PHP.

# Tools
- [x] VS Code
- [x] PhpMyadmin

# Features
* Login for Admin
* Admin Page
* Live Search
* automatically background changer
